Sign in system

A responsive web application that captures user data with real-time validation and displays it dynamically using browser storage.

Features
- Smart Validation: Real-time input checking using `blur` events.
- Data Persistence: Saves and retrieves user data via `localStorage` (JSON).
- Dynamic UI: Renders user information dynamically on the result page.
- Adaptive Design: Fully responsive layout for Mobile, Tablet, and Desktop.
- UX Focused: Submit button is disabled until all forms are valid and terms are accepted.

Tech Stack
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Framework: Bootstrap 5 (RTL)
- Storage: Browser LocalStorage API

Project Structure
- `index.html` Registration Form.
- `info.html` Data Display Page.
